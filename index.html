Blandcss

Blandcss is a small set of stylesheets which meant to be used as a foundation. It is very neutral on its own (hence, the name) and may require some modification according to your tastes. Though vanilla will still work too.

Blandcss is in the unlicense. Feel free to use and/ or abuse

Typography

All elements except small, and sup/sub use rem for size. So they change depending on the font-size of the :root.

The font-size of :root is also flexible. It use clamp() to create fluid typography.

Scaling

Blandcss use SASS to generate sizes. The default scale is 1.2. And the base size is the minimum font size which is 1rem. The calculations is like this:

  • The size of .smaller is equal to 1rem * 1.2-2
  • small is 1rem * 1.2-1
  • p is 1rem * 1.20
  • h6 is 1rem * 1.21
  • h5 is 1rem * 1.22
  • h4 is 1rem * 1.23
  • And so on until h1
Spacing

Blandcss use margin-top to spaced out block elements. Common block elements such as p, img, pre et cetera have margin-top the same size as .small or $base * $scale^-1. However, the margin between paragraph and other paragraph is based on .smaller.

Heading element from h6 to h4 have larger margin-top equal to the size of h4, while h3 to h1 have h3 size margin-top.

Centered Image

<img class="img-center" src="...">
Centered figure with image
<figure class="img-center">
  <img class="img-center" src="...">

</figure>

Floating image for the retard

Left floated image is possible for both img and figure with image by adding class img-left. But you need to make sure the content beside it is large enough to prevent it from collapsing. Its maximum width and height is capped at 50% viewport width and height.

Table

The table must use thead, tbody, tfoot accordingly. It is also possible to have stripped table by adding .stripped class to table elements.

Skins 1kg $100
Bones 2kg $120

Total $1125
